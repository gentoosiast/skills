# Agent skills

Agent skills, distilled from how I actually work.

## Skills

| Skill | What it does |
|-------|--------------|
| [`socratic-tutor`](./socratic-tutor) | Teaches a technical topic by making you reason it out instead of handing over answers — probing questions, escalating hints, docs-reading practice. |
| [`prep-talk`](./prep-talk) | Prepares a talk end to end in *your* voice — interviews you for a speaker profile, finds the angle, pins the vocabulary, builds a timed outline, reviews and fact-checks before the stage. |
| [`challenge-talk`](./challenge-talk) | Pressure-tests a talk's idea before you build it — steelman, then depth / audience relevance / audience fit, landing on a verdict: go, reframe, narrow, or drop. |
| [`content-review`](./content-review) | Final read-through of any draft — a sound pass for AI tells and machine rhythm, an argument pass for whether the claims hold, a register pass. Concrete notes, not vibes. |
| [`fact-check`](./fact-check) | Verifies technical claims against primary sources — docs, source, changelogs, reproducible examples — and marks what couldn't be verified instead of hiding it. |

## Install

With [`npx skills`](https://github.com/vercel-labs/skills) — pick the skill from the prompt:

```sh
npx skills add ArchieSA/skills
```
