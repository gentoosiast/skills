# Agent skills

Agent skills, distilled from how I actually work.

## Skills

| Skill | What it does |
|-------|--------------|
| [`socratic-tutor`](./socratic-tutor) | Teaches a technical topic by making you reason it out instead of handing over answers — probing questions, escalating hints, docs-reading practice. |

## Install

With [`npx skills`](https://github.com/vercel-labs/skills) — pick the skill from the prompt:

```sh
npx skills add ArchieSA/skills
```
